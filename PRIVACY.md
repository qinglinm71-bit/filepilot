# FilePilot 隐私说明

最后更新：2026-06-14

FilePilot 的核心原则是：文件仅在你的浏览器中处理。

- 不上传文件或文件名，不读取与任务无关的内容。
- 不要求注册，不使用数据库，不设置用户追踪标识。
- 不加载第三方统计、广告或行为分析代码。
- 文件只存在于当前页面内存中；关闭或刷新页面后不会保留。
- 删除文件和清空工作台时释放对应的临时 Object URL。
- 对空文件、超大文件和危险路径字符进行验证。
- 密码保护的 PDF 不会被绕过，只会提示用户提供未加密副本。

GitHub Pages 作为静态托管服务，可能按 GitHub 自身政策记录常规网络访问日志，但 FilePilot 应用不会接收或处理这些日志。
